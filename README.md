# 쏘쓰

카페/베이커리 사장님을 위한 네이버 플레이스 진단 기반 콘텐츠 운영 서비스, 쏘쓰

## 프로젝트 구성

이 저장소는 쏘쓰 클라이언트 애플리케이션을 모아놓은 모노레포입니다.

```
ssoss-client/
├── README.md          # 저장소 소개 및 프로젝트 구성
└── ssoss_flutter/     # Flutter 모바일 앱 (iOS, Android)
```

### ssoss_flutter

Flutter 앱 프로젝트는 [`ssoss_flutter/`](./ssoss_flutter/) 폴더에 있습니다.

자세한 프로젝트 설명은 [`ssoss_flutter/README.md`](./ssoss_flutter/README.md)를 참고하세요.
